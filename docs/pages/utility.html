If the content gets too long, an ellipsis will be placed and the text is cut off. Add a title attribute to display the whole text on mouseover.

Visibility

The ShowCarUI library provides some utility classes to show/hide elements based on the current viewport (media queries).

IMPORTANT: Try to use these on a limited basis and avoid creating entirely different versions of the same site. Instead, use them to complement each device's presentation.

S Viewport M Viewport L Viewport XL Viewport
.sc-visible Visible Visible Visible Visible
.sc-visible-xl Visible
.sc-visible-l Visible Visible
.sc-visible-m Visible Visible Visible
.sc-visible-s Visible Visible Visible Visible
.sc-hidden Hidden Hidden Hidden Hidden
.sc-hidden-xl Hidden
.sc-hidden-l Hidden Hidden
.sc-hidden-m Hidden Hidden Hidden
.sc-hidden-s Hidden Hidden Hidden Hidden
