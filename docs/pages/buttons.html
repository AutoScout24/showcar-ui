Buttons

ShowCar uses two types of buttons, .sc-btn-bob and .sc-btn-ross.

Use the button classes on an <a>, <button>, or <input> element.

.sc-btn-block adds block style to a button, which stretches the button to full width.

a .sc-btn-ross a .sc-btn-ross
