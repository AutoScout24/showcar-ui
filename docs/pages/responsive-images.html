Responsive Images

The way to go for responsive images in Showcar is the html <picture> tag. Unfortunately, it is not yet sufficiently supported by the browsers. Another, more supported way, is the srcset attribute of the <img> tag. In any case, we included picturefill to polyfill the missing functionality for unsupported browsers.
