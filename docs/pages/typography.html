Typography

ShowCars default font-family is Source Sans Pro. Please use the predefined classes to increase or decrease the font size. You should not define explicit font sizes. To use the Source Sans Pro font, you don't have to do anything.

Headlines

ShowCarUI provides responsive font sizes for headlines h1 - h5, you simply need to add the class .sc-font to the corresponding tag. To see the difference between small and larger viewports, simple resize the browser and see the following example.

viewport < M (768px) viewport ≥ M (768px)
h1.sc-font XL (24px) XXL (32px)
h2.sc-font L (20px), bold XL (24px), bold
h3.sc-font M (16px), bold XL (24px)
h4.sc-font default L (20px), bold
h5.sc-font default M (16px), bold
