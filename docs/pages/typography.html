Typography

ShowCars default font-family is Source Sans Pro. Please use the predefined classes to increase or decrease the font size. You should not define explicit font sizes. To use the Source Sans Pro font, you don't have to do anything. It is loaded with the showcar-ui library using a technic called font-face observer.
