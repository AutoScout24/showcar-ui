Variables

If you need to write some custom scss code there are several scss variables you should use.

To use the variables in your scss code you just need to import the specific file based on your directory structure: 


@import "variables";

Spacings

There are several variables that should be used for the spacing inside and between elements. Use the variables only for the spacing and not for other properties like e.g. font-sizes.

variable value
$XS 4px
$S 8px
$M 12px
$L 16px
$XL 20px
$XXL 28px
$XXXL 40px

Viewports

All responsive elements that are based on media queries use the following breakpoints. These dimension should be usefull in most of the cases.

Use the following variables instead of variables previously known from showcar-styles ($mobile, $tablet, $desktop)

variable value size, from/to
$viewportXS 0 0 - 320px
$viewportS 321px 321px - 767px
$viewportM 768px 768px - 1023px
$viewportL 1024px 1024px - 1099px
$viewportXL 1100px 1100px - 1269px
$viewportXXL 1270px 1270px - ∞
