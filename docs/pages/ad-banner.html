Ad Banner

The ad banner (fireplace) consists of three div elements which must be placed inside the sc-content-container div. There are also the following additional classes which need to be integrated as needed in the sc-content-container div: sc-skyscraper-left, sc-skyscraper-right and sc-superbanner

Be sure to add an overflow-x: hidden to the body tag, so that there is no possibility of horizontal scrolling!
